# XXSwitch 文档

本目录保存 XXSwitch 的架构决策和历史设计资料。当前产品行为以根目录 [README](../README.md)、[领域术语](../CONTEXT.md) 和代码为准。

## 架构决策

`adr/` 记录不容易逆转且需要保留取舍背景的决策，包括供应商状态与自动停用、Pi 路由，以及跨客户端 Skills 管理。

ADR 描述的是决策约束，不用于罗列界面字段或发布说明。功能变化若不改变原决策，只更新 README 和领域术语即可。

## 界面设计

`design/` 中的图片是早期设计与迭代快照，用于追溯视觉方向，不代表当前界面。当前供应商编辑器、使用统计、请求列表和路由页面请以运行中的应用为准。
